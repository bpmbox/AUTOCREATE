- **フォント**: Inter, Noto Sans JP, system-ui

- **本文**: body(1rem), small(0.875rem), caption(0.75rem)
- **行間**: 1.5倍（読みやすさ重視）

- **Mobile**: 320px - 767px

## 📱 画面遷移フロー

### 認証フロー
```
ランディング → ログイン → ダッシュボード
    ↓              ↓
会員登録 → メール認証 → プロフィール設定
    ↓
ソーシャル連携 → 本人確認 → 完了
```

### プロフィール管理フロー
```
ダッシュボード → プロフィール表示 → 編集
                    ↓
            本人確認 → 書類アップロード → 審査結果
```

### 予約システムフロー
```
ダッシュボード → 予約カレンダー → 日時選択 → 確認 → 完了
                    ↓
            予約管理 → 変更・キャンセル → 確認メール
```
