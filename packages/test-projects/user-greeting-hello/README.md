
## 📊 処理フロー
1. ユーザーからの挨拶を検出
2. 現在時刻に基づいた適切な挨拶を生成
3. システム情報とサービス案内を含む応答を作成
4. Supabaseのchat_historyテーブルに自動送信
5. 処理完了通知
